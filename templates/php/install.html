一、apt命令



        sudo apt install php7.0-cli php7.0-fpm php7.0-common php7.0-curl php7.0-gd php7.0-json \
        php7.0-mbstring php7.0-mcrypt php7.0-readline php-igbinary


二、各模块含义


模块名称 用途
php7.0-cli php7.0命令行，用于命令行脚本执行
php7.0-fpm php7.0 fpm模块，用于web请求处理
php7.0-common php7.0通用模块
php7.0-curl curl扩展，用于网页／网站资源的下载
php7.0-gd php图形处理扩展，可创建／处理图形文件
php7.0-json json扩展，用于json数据的编码／解码
php7.0-mbstring 用于替换iconv的字符编码模块
php7.0-mcrypt 加解密的模块
php7.0-readline 主要用于php-cli，支持命令历史切换，如php -a时
php-igbinary igbinary扩展，性能远高于自带的serialize/unserialize函数

三、验证安装


1、查看php版本

php -v

2、查看php模块

php -m

3、查看php-fpm进程

ps -ef | grep php-fpm
