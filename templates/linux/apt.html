linux软件安装方式


  • 源代码编译方式
    • 优点：可实现更细微的选项配置；可自行选择安装某软件的特定版本
    • 缺点：需手工安装依赖模块;需手工配置安装后的管理脚本（如服务软件包的开机自启动）
  • 软件包模式
    • 优点：自动安装依赖模块；自动加入服务管理体系
    • 缺点：只能安装软件源支持的特定版本（该版本可能很旧）

ubuntu软件包管理高级命令


命令 功能
sudo apt update 更新可用包列表，若有系统升级包/新版本软件包会提醒
sudo apt upgrade 升级系统／软件包
sudo apt install package_name 安装指定软件包，会自动安装依赖包
sudo apt remove package_name 安装指定软件包，会自动安装依赖包
sudo apt autoremove 自动删除所有不被依赖的无用软件
sudo apt list php7.0* 查看软件源中所有以php开头的软件包
sudo apt search php 搜索描述中包含php的所有软件包
sudo apt show php7.0-cli 显示包详情

ubuntu软件包管理低级命令


命令 功能
dpkg -i package_file 安装／更新已经下载的软件包package_file
dpkg -l 列出已安装的软件包
dpkg -s 显示包详情
dpkg -S file_name 查看文件属于哪个软件包
