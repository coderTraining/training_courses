命令 功能
type 显示命令的类型，有4种类型：可执行程序、内置命令、Shell函数和别名
which 显示可执行程序的位置
whatis 显示一条命令的简述
help 获取内置命令的帮助
--help 可执行程序的帮助
man 查看命令的手册
info 查看带链接的手册
? 显示帮助命令
PageUp/Backspace 返回上一页
PageDown/Spacebar 翻到下一页
n 显示下一节点
p 显示上一节点
u 返回父节点
Enter 进入当前链接
q 退出
