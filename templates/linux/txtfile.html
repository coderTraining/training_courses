命令 功能
file 查看文件类别
wc 字、行、字符、字节计数
wc -l file 查看文件行数
wc -w file 查看文件词数
wc -c file 查看文件字节数
wc -m file 查看多字节文本字数
tail 查看文件末尾部分
tail -n file 查看文件最后n行，默认为20行
tail -f file 查看文件的持续输出，常用语查看实时日志
head 查看文件开头部分
head -n file 查看文件开头n行，默认为20行
cat 连接和打印文件
cat file 输出文件内容
cat file1 file2 将多个文件内容连接并输出
