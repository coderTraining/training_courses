命令 意义
vim file，vim file1 file2 ... 打开单个文件，打开多个文件(-o 多个文件水平排列，-O 多个文件垂直排列)
open file，split file，vsplit file 在当前窗口中打开一个新文件，水平切分新窗口打开文件(sp)，垂直切分新窗口打开文件(vsp)
:bn，:bp，:bd，:ls 切换到下一文件，切换到上一文件，关闭当前窗口，显示所有打开文件列表(或:args)
Ctrl+ww，Ctrl+wj，Ctrl+wk 移动到下一窗口，移动到下方的窗口，移动到上方的窗口
:wa，:qa，:only 保存所有文件，关闭所有窗口，仅保留当前窗口
