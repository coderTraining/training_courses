命令 意义
:q，:w，:wq，:x，:w!，:q! 退出，保存，保存并退出，保存并退出(文件没修改的话不会修改时间戳)，强制保存，强制退出并忽略所有修改
:w file 另存为file
:r file 读取file内容放到当前行下面
:1,10 co 20 将1至10行复制到第20行之后
:1,10 m 20 将1至10行移动到第20行之后
:10 pu 将剪切板中的内容粘贴到第10行后
:!command 执行外部命令，如:!ls 列出当前目录下文件
:= 显示文件总行数
:help，:help d，:help 'number'，Ctrl+]，Ctro+o/t 帮助，d命令的帮助，number选项的帮助，帮助中的链接点击方法，从链接内容返回
