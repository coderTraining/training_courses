一、安装



        sudo apt install mysql-client mysql-server mysql-common php70-mysql


二、额外配置


修改/etc/mysql/mysql.conf.d/mysqld.cnf,在[mysqld]下增加一项

character_set_server = utf8



三、重启mysql


sudo service mysql restart


四、验证安装和配置


1、验证编码配置

show variables like '%character%';

2、验证php的mysql扩展安装

php -m | grep mysql
